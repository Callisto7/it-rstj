Отдел научных разработок МТУСИ (Московский технический университет связи и информатики)

Российский отраслевой университет в области информационных технологий, телекоммуникаций, информационной безопасности.

О проекте

PVision — это платформа для аналитики в реальном времени, которая с помощью нейросетей и камер видеонаблюдения отслеживает использование средств индивидуальной защиты (маски, жилеты).

Платформа предоставляет заказчикам инструменты для подключения камер, настройки сценариев мониторинга и получения аналитики.

Проблема

Заказчик обратился с рядом проблем / потребностей:

  • Отсутствует единый инструмент для мониторинга доступности банковских сервисов.
  • Необходимо оперативное выявления проблем.
  • Есть потребность в анализе пользовательских мнений из соцсетей, но нет сервиса по его анализу.
  • Необходима наглядная визуализация статистики и трендов.
  • Необходим механизм, который будет предоставлять обратную связь от пользователей.

Задача

Разработать сервис для мониторинга доступности веб-ресурсов 29 крупнейших банков России в реальном времени.

Система должна анализировать мнения пользователей из социальных сетей с помощью нейросетей, предоставлять комплексный дашборд с визуализацией данных и позволять отправлять отчеты о проблемах.

Технологии

React.js, JavaScript, Chakra UI, Chart.js, Axios, React forms, React router dom, Docker, Nginx

Решение

В ходе работы над проектом реализовано:

  • Интерактивный каталог банков с цветовой индикацией статуса.
  • Детальная страница отчета с историей доступности ресурса.
  • Комплексный дашборд с фильтрами и настраиваемыми виджетами.
  • Визуализация анализа соцсетей (графики).
  • Система цветовых индикаторов статусов доступности.
  • Механизм отправки репортов с прикреплением скриншотов.
  • Адаптивный дизайн для разных устройств.
  • Кэширование данных для повышения отзывчивости.

В процессе разработки:

  • Использовался компонентный подход к разработке интерфейса.
  • Внедрен регулярный code-review для поддержания качества кода.
  • Произведена оптимизация производительности критических компонентов.
  • Реализован автоматизированный деплой через CI/CD.
